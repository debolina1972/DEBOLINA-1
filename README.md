# Student Performance Prediction System

## 📌 Project Overview
The Student Performance Prediction System is a Machine Learning-based project that predicts student academic performance using factors such as attendance, study hours, previous marks, and assignment scores.

The system helps educational institutions identify weak students early and take corrective actions to improve academic outcomes.

---

## 🎯 Objectives
- Predict student academic performance
- Identify students at risk of failing
- Improve teaching strategies using data analysis
- Support data-driven decision making
